## Demo description

Hover-based interactions can be simple yet elegant. Using the 

`onCellHoverIn`

 and 
`onCellHoverOut` events, the calendar dynamically displays an "Add event" button when users hover over day cells, providing an intuitive way to create new events with minimal interface clutter.

For customizing the cell content appearance, the `renderCalendarDayContent` is the perfect candidate, as it holds the basic structure of the day cell.
